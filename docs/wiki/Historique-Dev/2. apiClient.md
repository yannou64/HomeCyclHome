# apiClient

Il s'agit de créer une instance Axios partagée par toutes les features. Cela permet de centraliser deux comportements communs à toutes les requêtes HTTP :

- **`baseURL`** : le chemin de base de l'API — les services n'écrivent plus que `/auth/login` au lieu de l'URL complète
- **`withCredentials: true`** : indique au navigateur d'envoyer automatiquement les cookies lors des requêtes. C'est indispensable pour que le cookie HttpOnly contenant le JWT parte avec chaque appel API

```ts
export const apiClient = axios.create({
    baseURL: import.meta.env.VITE_API_URL,
    withCredentials: true,
});
```

Il est installé dans `shared/services/` car toutes les features l'importeront. La règle de base est que les features doivent être indépendantes entre elles — une feature ne doit pas importer depuis une autre feature.
